The Eye In The Sky 7th Inter IIT Tech Meet

As a part of this project, I build a Satellite image segmentation model. I used a modified version of U-net. The key observation that we found was that neural network was unable to learn the features or distinguish between some similar features (say for example roads and railways has almost similart patterns) when only the input image was feeded to the model architecture. So I did feature engineering and tried to generated new feature like Edge feature, Texture feature etc using image processing techinques like local binary transform, Sobel edge detection etc and then feeded these features along side the image.

Implementation of FFT on FPGA Part of Digital System course

The DFT is a linear transformation of a vector Xn(time domain) to the vector Xm(coefficients of the components sinusoids of the time domain signal). DFT requires O(N2) while FFT uses the precomputed results and require O(Nlog(N)) time. Implemented 8 point Radix 2 FFT having Bit Reveral module, Butter Fly unit, Twiddle Factor Generator and Control Module.
